# Data Science Interview Crash Course

A collection of Jupyter notebooks for data science interview preparation.

## Notebooks

| # | Topic | Description |
|---|-------|-------------|
| 01 | Pandas | DataFrame operations, merging, groupby, pivot tables |
| 02 | NumPy | Array operations, broadcasting, linear algebra |
| 03 | SQL | Queries with SQLite - JOINs, CTEs, window functions |
| 04 | Statistics | Probability, distributions, hypothesis testing, A/B testing |
| 05 | Machine Learning | Scikit-learn classifiers, regression, cross-validation, pipelines |
| 06 | Data Cleaning | Missing values, outliers, encoding, scaling, feature engineering |

Each notebook includes teaching sections followed by practice questions with collapsible answers.

## Setup

```bash
uv sync
uv run jupyter lab
```

## Requirements

- Python 3.12+
- uv package manager
